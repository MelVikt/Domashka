Робота з блоковими та інлайновими елементами

Це параграф, що знаходиться всередині блокового контейнера div. Блокові елементи займають всю ширину доступного простору.

Ще один блоковий елемент, де ми використовуємо інлайновий елемент для виділення частини тексту всередині абзацу.

Ще один приклад з інлайновими елементами. Текст тут містить текст, виділений червоною фарбою для акценту.

Ось посилання a, яке розташовано всередині блокового елемента.

Цей блок містить жирний текст та курсивний текст для підкреслення важливих частин контенту.

Ось інлайнове поле для вводу (яке не займає всю ширину сторінки, а розташовується на одному рядку з іншими елементами): можна написати текст після поля для вводу.
